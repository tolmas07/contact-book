ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА

Для корректной работы приложения убедитесь, что у вас установлен Node.js.

Шаги запуска:

1. Установка зависимостей:
   Откройте терминал в папке проекта и выполните команду:
   npm install

2. Запуск в режиме разработки:
   Выполните команду:
   npm run dev

3. После запуска в терминале появится ссылка (обычно http://localhost:5173).
   Откройте её в браузере.

4. Сборка проекта (для продакшена):
   Выполните команду:
   npm run build

Дополнительная информация:
- Весь исходный код находится в папке /src.
- Иконки и графические ресурсы перемещены в /src/assets/icons.
- Стили написаны на SCSS и находятся в /src/styles.
- Логика приложения реализована на TypeScript в /src/scripts.
